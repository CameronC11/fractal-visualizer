# Fractal Visualizer
This is a simple vizualer for the Mandelbrot Set fractal.
Controls: Scrollwheel up/down for zooming in/out respectively.
How to run: 
1. Create 'build' folder in root directory
2. Run command 'cmake .' in build directory (or 'cmake build' in root dir)
3. Run 'cmake --build .' in build directory ('cmake --build build' in root)
4. Locate and run executable file
